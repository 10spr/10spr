#49. JS indexOf() Method

The indexOf() method searches an array for an element value and returns its position. So I'll try searching for the index of "1975" in the array years.

Array.indexOf() returns -1 if the item is not found. If the item is present more than once, it returns the position of the first occurrence.

#50. JS lastIndexOf() Method

Array.lastIndexOf() is the same as Array.lastIndexOf(), but returns the position of the last occurrence of the specified element.

This means, if there are two 1975 elements in the years array, it would return the index of the second one instead of the first.

#51. JS find() Method

The find() method returns the value of the first array element that passes a test function.

JavaScript always works from left to right, this is why 2001 is the first element that is bigger than 1500.

#52. JS findIndex() Method
